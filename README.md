# API Clean Architecture

## Como executar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o servidor em modo de desenvolvimento:

```bash
npm run dev
```

O comando `npm run dev` executa `nodemon src/infrastructure/api/server.ts` e reinicia automaticamente quando houver alterações nos arquivos.

## Como rodar os testes

```bash
npm test
```
